# Contributing to Veilfire Chat

Thank you for your interest in contributing to Veilfire Chat!

## How to contribute

- Fork the repository.
- Create a feature branch for your changes.
- Make your edits in small, focused commits.
- Open a pull request with a clear description of the motivation and the changes.

## License for contributions

By submitting a pull request, you agree that your contributions are licensed
under the same Apache-2.0 license as this project.

If you include third-party code, please make sure that it is compatible with
Apache-2.0 and clearly document any additional licensing requirements.
